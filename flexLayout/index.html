    容器的属性
  1. flex-direction : 决定项目的排列方向.
  2. flex-wrap : 决定项目是否可换行.
  3. flex-flow : 前两个属性的合体，默认row nowrap.
  4. justify-content : 决定项目在主轴的对齐方式.
  5. align-items : 决定项目在交叉轴的对齐方式.
  6. align-content : 决定项目在多条主轴上的对齐方式.
    项目的属性
  1. order : 决定项目的排列位置数值越大越靠后.
  2. flex-grow : 决定项目的放大比例.
  3. flex-shrink : 决定项目的缩小比例.
  4. flex-basis : 决定项目的固定尺寸.
  5. flex : 前三个属性的合体 默认 0 1 auto.
  6. align-self : 决定单个项目的对齐方式.
